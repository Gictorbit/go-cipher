# Gocipher
Caesar cipher implementation written in go

## example
```bash
./sezarapp encrypt -txt="Hello World" -shift=15

./sezarapp decrypt -txt="Wtaad Ldgas" -shift=15

./sezarapp crack -txt="Wtaad Ldgas"
```
## make
build for linux
```
make build
```
Compile for most common operating systems
```
make compile
```
